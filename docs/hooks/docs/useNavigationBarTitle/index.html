useNavigationBarTitle

动态更新导航栏属性.

提示

只有当前页面显示时才会更新导航栏标题

用法#

import { useNavigationBarTitle } from '@tarojsx/hooks'
const Demo = ({ article }) => {
useNavigationBarTitle(article.title)
}
